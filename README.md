# WHT: What's Happening in Tech

Welcome to the **WHT** repository!

## The Concept
WHT is a tailored platform designed specifically for Gen Z and international students to bridge the severe gap between traditional hiring advice and the modern reality of the tech job market. The hiring landscape is fundamentally broken, saturated with applicant tracking systems (ATS) and brutal algorithmic filters. Traditional advice like "just apply on the portal" simply doesn't work anymore. 

The mission of WHT is to empower early-career tech engineers and professionals by decoding the system. The platform offers definitive playbooks on:
- Utilizing **Agentic AI** to automate cold outreach and job searches.
- Formulating the perfect resume engineered to bypass an ATS.
- Mastering inbound profile SEO and strategic networking.

## The WHT LinkedIn Ecosystem
This platform scales seamlessly alongside the **WHT LinkedIn Page**, serving as a community engagement funnel. While the LinkedIn page rapidly builds community by sharing live market trends, layoff awareness, visa shifts, and weekly tech updates, this website acts as the permanent, heavy-hitting hub. It stores long-form detailed guides, custom AI workflows, and the official Job Search Playbook.

## System Architecture
The platform features a highly responsive, custom-styled brutalist UI designed for immediate impact.
- **Framework:** React + Vite
- **Routing:** React Router DOM
- **UI & Styling:** Custom Vanilla CSS (Sunset Brutalism Aesthetic, Floating Dynamic Sidebars)

## Running the Platform Locally
If you want to boot up the website locally and test the UI:

1. Clone the repository:
   ```bash
   git clone git@github.com:JyothsnaVellanki22/WHT.git
   cd WHT
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
